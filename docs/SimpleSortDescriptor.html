columnType :number

How this column is to be sorted. If SimpleDataTableUtils.COLUMN_TYPE_INFER, will attempt to convert values to numbers prior to running the sort comparison; values that cannot be converted will be compared as strings, and will be sorted under those successfully converted to numbers. If SimpleDataTableUtils.COLUMN_TYPE_TEXT, all values will be compared as strings only.
