Class: SimpleFilterDescriptor

SimpleFilterDescriptor(columnIndex, compareValue, operationopt, columnTypeopt)

Generic implementation of FilterDescriptor. Filters cells for the given columnIndex by comparing their values to the given compareValue using the given operation. The values for individual cells are determined based upon columnType.

Operation is a string, and must be one of the following:

  • '=' (Equals)
  • >' (Greater Than)
  • '<' (Less Than)
  • '>=' (Greater Than Or Equal To)
  • '<=' (Less Than Or Equal To)
  • '!=' (Not Equals)
  • '~' (Contains, Ignore Case)
  • '~~' (Contains, Case Sensitive)
